Disclaimer: This dashboard aggregates published asset allocation frameworks and model portfolio views from Ned Davis Research, BlackRock Target Allocation ETF Models, and BCA Research. Signals represent each firm's tactical tilt direction relative to a neutral strategic benchmark. This is not investment advice. Overweight/underweight designations reflect relative positioning, not absolute return forecasts.

Data pipeline: python/update_data.py runs via GitHub Actions on the 1st of each month, writes data/allocations.json, and commits back to main. GitHub Pages serves the updated dashboard automatically.
